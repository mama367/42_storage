<기획의도> 
- 2조 안에서의 취미 공유하는 과정에서
- 한국 문화, 정체성을 상징하는 요소 중 K-POP에 집중 
- 쇼핑몰과 커뮤니티를 아우르는 아카이빙 플랫폼을 제작
- Ex. 참고 사이트: 위버스, 번개장터

<기술스택> 
- Language: Java, JavaScript, Ajax
- Framework: Spring-boot
- DB: MYSQL
- IDE: IntelliJ
- Templete Engine: Thymeleaf
